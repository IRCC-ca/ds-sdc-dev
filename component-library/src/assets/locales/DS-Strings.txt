Buttons.Intro	Buttons trigger an action or navigate to another page. They have multiple styles and configurations for various needs. They call attention to actions the user should take to progress through a flow.	Les boutons déclenchent une action ou permettent de naviguer vers une autre page. Ils ont de multiples styles et configurations selon les besoins variés. Ils attirent l’attention sur les actions que l’utilisateur doit effectuer pour progresser dans un flux.

Buttons.PrimaryBtnText	Primary button	Bouton principal
Buttons.PrimaryText	Primary buttons are used to highlight the most important actions in any experience. Avoid using more than one primary button in a section or screen whenever possible.	Les boutons principaux sont utilisés pour mettre en évidence les actions les plus importantes dans toute expérience utilisateur. Éviter autant que possible d’utiliser plus d’un bouton principal dans la même section ou le même écran.
Buttons.SecondaryHeading	Secondary	Secondaire
Buttons.SecondaryIBtnText	Secondary button	Bouton secondaire
Buttons.SecondaryText	Secondary buttons are used most often and are used when less visual attention is required on an action. They can also be used alongside a primary action if there are two options to present to the user that require a different hierarchy.	Les boutons secondaires sont les plus utilisés et ils sont utilisés lorsqu’une action requiert moins d’attention visuelle. Ils peuvent également être utilisés parallèlement à une action principale s’il y a deux options à présenter à l’utilisateur et que ces options nécessitent une hiérarchie différente.

Buttons.PlainBtnTexts	Plain buttons	Boutons simples
Buttons.PlainText	Plain buttons can be used for actions that require less attention than a primary or secondary button. They are also useful in situations where the UI is more compact. They are not to be used as links. Use the link text styles for those scenarios.	Les boutons simples peuvent être utilisés pour des actions qui nécessitent moins d’attention que celles des boutons principaux ou secondaires. Ils sont également utiles dans les situations où l’interface utilisateur est plus compacte. Ces boutons ne doivent pas être utilisés comme des liens. Utiliser les styles de texte des liens dans ces situations.

Buttons.SpacingGroupsHeading	Spacing for button groups	Espacement des groupes de boutons
Buttons.SpacingGroupsText	A button group is a set of buttons that have a contextual relationship. To create consistency with how these actions are visually related, consistent spacing rules should be followed. Large button groups should have 16 pixels between buttons and small button groups should have 12 pixels between buttons.	Un groupe de boutons est un ensemble de boutons qui ont une relation contextuelle. Pour créer une cohérence dans la façon dont ces actions sont liées visuellement, des règles cohérentes d’espacement doivent être suivies. Pour les grands groupes de boutons, il doit y avoir un espacement de 16 pixels entre les boutons. Pour les petits groupes de boutons, il doit y avoir un espacement de 12 pixels entre les boutons.

Il est personnalisé pour l'expérience du client et de l'agent et permet aux équipes de créer efficacement des services numériques d'immigration cohérents.
